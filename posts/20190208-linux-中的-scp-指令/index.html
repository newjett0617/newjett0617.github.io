Linux 中的 scp 指令

2019-02-08 · Jett
Table of Contents
  • remote -> local
  • local -> remote

scp 指令是在網路上主機之間複製文件的指令，使用 ssh 進行身份驗證、安全地數據傳輸檔案

remote -> local#

scp user@remote_host:remote_file local_file

local -> remote#

scp local_file user@remote_host:remote_file
