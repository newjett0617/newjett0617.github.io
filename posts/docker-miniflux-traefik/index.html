[Docker] 自架 RSS, 使用 Miniflux 搭配 Traefik

最近因為 Inoreader 的免費訂閱數量快到上限了，不想掏錢購買服務，在思考有什麼工具來解決我訂閱數量的問題，搜尋 self hosted rss 就有一堆文章及工具提供選擇，於是我挑了 Miniflux 來使用

安裝 Miniflux

使用 docker-compose 來進行安裝， repository 中有提供 basic.yaml 範例當作使用，就不多贅述了

搭配 Traefik

repository 中也有提到如何與 Traefik 整合的做法，traefik.yml ，只不過他提供的作法無法滿足我的需求，稍微修改成符合我需求的樣子 docker-compose.yml

值得一提，我的情況是需要新增 traefik.http.services.miniflux.loadbalancer.server.port=8080 這個 label 才能把流量轉導到 miniflux 容器內
