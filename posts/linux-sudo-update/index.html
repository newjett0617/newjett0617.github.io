[Linux] 安全性更新 Sudo

最近看到 這篇 說到，在預設 sudo 組態的情況下，非擁有 root 權限的使用者能夠透過 sudo 來提升權限而獲得 root 的權限，只要能登入主機的都可以獲得 root 的權限

也看到 這篇 有提供的指令可以檢查看看是否受到此漏洞的影響，自己檢測一下發現居然不是 usage: 開頭，看來我要來更新套件了

我是參考 這篇 的做法，使用 apt 方式更新套件

# apt update
# apt list --upgradable
# apt --only-upgrade install <PackageName>

在 update 後可以看看有哪些套件是可更新的，剛好有看到 sudo 這個套件，那就來更新吧
